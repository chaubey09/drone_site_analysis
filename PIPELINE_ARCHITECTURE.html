🚇 Metro Construction Progress Pipeline

Bi-Monthly Drone Video Analysis → Automated Progress Report
📹 INPUT  |  Drone video ~30min @ 30fps (≈54,000 frames) + optional GPS SRT file
~15-25 min
01
Preprocessing & Frame Extraction
01_preprocess.py

Operations

Video stabilization (optical flow) Extract 1 frame / 2 sec Blur detection (Laplacian) Duplicate filtering (MAD) DJI SRT GPS parsing

Output

~900 clean JPEG frames frame_metadata.json {lat, lon, alt, heading} per frame
54,000 frames→ ~900 quality frames (98% reduction)
~10-30 min
02
Georeferencing & Orthomosaic
02_georeference.py

Operations

GPS → pixel mapping (equirectangular) Frame footprint estimation from altitude + FOV Weighted frame blending SIFT feature stitching (fallback) GeoTIFF export (rasterio)

Output

orthomosaic_{period}.png orthomosaic_{period}.tif (georeferenced) stitch_metadata.json
Resolution: 5cm/px Full site top-down view
~5-20 min
03
Semantic Segmentation
03_segment.py

11 Construction Classes

Concrete structure Steel rebar Formwork Earthwork Equipment Material stockpile Vegetation Road surface Water body Workers

Two Modes

Heuristic (HSV + texture) — works instantly Mask2Former / SAM — fine-tuned deep model

Output

segmentation_mask_{period}.png segmentation_overlay_{period}.jpg class_areas_{period}.json (m² per class)
~2-5 min
04
Change Detection
04_change_detection.py

Change Types Detected

🟢 New construction 🔴 Demolition / removal 🟠 Equipment movement 🔵 Earthwork progress

Method

SIFT homography alignment Pixel-level mask diff Morphological noise removal

Output

change_map_prev_curr.png change_heatmap_prev_curr.png comparison_prev_curr.jpg change_stats.json

Per Zone

Δ construction m² per zone Status: progressing / stalled / regressed
<1 min
05
Progress Metrics & KPIs
05_metrics.py

Computed Metrics

Overall completion % (weighted) Activity score (0-100) Per-zone completion % Δ construction m² per zone Alert detection (stalled / regressed)

Timeline Tracking

timeline_data.json (cumulative all periods) progress_metrics_{period}.json

Alerts Generated

⚠️ Stalled zone 🔴 Regression 🚀 Rapid progress
~1-2 min
06
Report Generation
06_report_generator.py

Report Sections

Executive Summary (KPIs) Before/After orthomosaics Segmentation overlays Change detection maps Zone progress table Timeline charts Alerts & observations

Output Formats

📄 PDF (ReportLab) 🌐 HTML (standalone, Chart.js)

Includes

Embedded base64 images Interactive timeline charts Color-coded zone table
📊 OUTPUT  |  progress_report_{period}.pdf + progress_report_{period}.html
